# Simple Collections Management Server

## Installation

Prerequisites:
- NodeJS 0.10
- MongoDB 2+
- Bower (nmp install -g bower http://bower.io/)

 `npm install`
 
 `bower install`

## Running The App

From the root of the project

 `node server.js`

Once running, the UI is accessible at:

[http://localhost:3000](http://localhost:3000)
